Media Controls

This slide contains 3 audio objects
that have been made visible at
runtime. They feature a player
control to start and stop each audio.

So you can play audios by animation
and synchronize them with other
animations.

You can make them visible for manual
user interaction as well.

And there is a third way to play an audio:
by executing a script when another
object is being clicked!
